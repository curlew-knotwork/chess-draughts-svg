# chess-draughts-svg

# DESCRIPTION

svg for laser cutting an 8x8 chess set and 10x10 checkers/draughts set
from 3 mm plywood.

# STATUS
Status = work in progress (7.12.2025)
not finished yet
I am working on the legs/stands for the chess pieces.

# NOTES

8x8 and 10x10 checker boards from https://commons.wikimedia.org/wiki/File:8x8_checkered_board_.svg and
https://commons.wikimedia.org/wiki/File:10x10_checkered_board.svg

8x8 Chess board and pieces.
10x10 Draughts board and pieces.
All board pieces drawn by hand using Inkscape

Lasercutting: hairline (red) is for burning. Black is for etching.
